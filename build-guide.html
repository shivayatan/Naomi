1

Preparation

  • • Clear a large, clean workspace with good lighting
  • • Have your tools ready (screwdriver, zip ties, etc.)
  • • Plug in your PSU but leave it switched OFF
  • • Touch the metal part of the PSU regularly to ground yourself
  • • Keep components in their anti-static bags until needed
2-4

Motherboard Assembly (Outside Case)

Step 2: Install CPU
  • • Open CPU socket lever
  • • Align triangle on CPU with socket
  • • Gently drop CPU in (no force)
  • • Close lever
Step 3: Install RAM
  • • Open RAM slots
  • • Align notch on RAM with slot
  • • Push down firmly until it clicks
  • • Do both sticks
Step 4: Install Cooler
  • • Apply thermal paste (if needed)
  • • Mount the 280mm AIO
  • • Connect pump power to AIO_PUMP header
  • • Connect radiator fans
5

Install Motherboard into Case

  • • Install I/O shield into the case (if not pre-installed)
  • • Place motherboard onto standoffs
  • • Screw it down (don't overtighten)
  • • Connect 24-pin power and 8-pin CPU power
6-7

Install GPU & Storage

Install GPU
  • • Remove PCIe slot covers
  • • Insert RTX 5080 firmly
  • • Screw it down
  • • Connect power cables (use native 12V-2x6 if possible)
Install Storage
  • • Install Samsung 9100 PRO in M.2 slot
  • • Screw it down
  • • Use the included heatsink if available
8-10

Cables & Final Steps

  • • Connect all power cables (24-pin, CPU 8-pin, GPU power)
  • • Connect front panel connectors (Power SW, Reset, LED, USB)
  • • Connect AIO pump and radiator fans
  • • Route cables neatly in the second chamber
  • • Double-check everything before powering on
Take your time. Rushing causes most mistakes.
When in doubt, refer to the manuals or ask here.
